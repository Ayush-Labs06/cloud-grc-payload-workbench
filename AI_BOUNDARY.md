# AI Boundary

## Allowed

- Explain concepts, mental models, syntax, and debugging strategy
- Generate theory notes before implementation
- Grade completed weekly work against the rubric
- Ask Socratic questions

## Disallowed

- Write scripts, configs, policies, or implementation code
- Fix bugs line by line
- Turn error messages into direct code changes

## When Stuck

Say: "Explain the concepts I should inspect and give me a debugging checklist, but don't solve it."

Do NOT say: "Fix this error and tell me exactly what to change."
